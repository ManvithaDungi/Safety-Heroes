# 🛡️ Safety Heroes

Safety Heroes is an interactive educational web application designed to teach children about safe and unsafe objects through a fun and engaging game.  
Players are presented with various real-world items and must decide whether each is **Safe** or **Unsafe**.  
The app also provides safety tips and explanations to help children learn and make better safety decisions.

## ✨ Features
- **Interactive Game** – Players identify if objects are safe or unsafe.
- **Educational Feedback** – Detailed explanations for correct and incorrect answers.
- **Accessibility Friendly** – ARIA labels, skip links, and keyboard navigation support.
- **Responsive Design** – Works across desktops, tablets, and mobile devices.
- **Navigation Bar** – Links to Home, Game, and Tips pages.
- **Image Fallback** – Placeholder images when an image fails to load.

---

## 🛠️ Tech Stack
- **Frontend Framework:** [React.js](https://react.dev/)
- **Routing:** [React Router DOM](https://reactrouter.com/)
- **Styling:** CSS (custom styles for components and pages)
- **Assets:** Local images for game items
